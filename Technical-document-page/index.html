Introduction

Sockets have traditionally been the solution around which most real-time chat systems are architected, providing a bi-directional communication channel between a client and a server contratry to polling.


Using sockets, a server can push messages to clients. Whenever you write a chat message, the idea is that the server will get it and push it to all other connected clients.
